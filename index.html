The site you are looking at right now is designed and developed by me. While I was learning flex and sass, I thought I could make my personal website by practicing these technology. The background video was modified using after effects and photoshop to play back and forth infinitely.

The site is made from holy grail model which allows its menu collapse with minimal disruption using flexible boxes.

Flex was great when it comes to centeralizing elements both horizontally and vertically. However, there were few problems when it came to compatibility with other browsers and devices. The problems involved around iOS devices as its chrome and other browsers are based off from Safari. Safari does not get its element height to 100% even though a user assigns it. I resolved this issue by using 100vh.

I've created Loading Screen using after effects to create the motion and photoshop to render it to gif form.

The site still has some bug on few browsers and devices, but I am still working on it. Please let me know if you find any weird bug you may observe!

Tree of Knowledge

UI|UX / Development | 2017
Download Style Guide PDF Link to the demo
Technology | Skills
  • UI | UX Design
  • JavaScript
  • NFC | iOS mobile app
  • HTML | CSS
  • Adobe Suite

Tree of Knowledge was exhibited in Digifest 2017. It is an immersive learning tool for children to gain knowledge about the world. Users pick up various shapes, from animals to stars, of wooden piece from tree structure, then place the piece on a stage to solve interesting questions.

My job on this project was to plan, lead and develop all technological parts. Near-field communication technology was chosen to initiate the page on an iPad app, then the app passed corresponding Javascript codes to the site to view the right page to the user.

As the most NFC frameworks are focused on android or apple app developments, I developed an iPad app which loads the website in Kiosk mode then passes Javascript code to load the corresponding page.

AutoBin

UI|UX / Design | 2017
Link to the invision demo
Technology | Skills
  • UI | UX Design
  • Prototype
  • Adobe suite

AutoBin is a recycling app designed for Canadians who have different biweekly/weekly waste collection schedule. The schedule varies depending on where the user live. The app not only notify the user the correct schedule but also tracks how much he or she has correctly placed waste into a correct container. Depending on how much the user have correctly put waste in, it transfers into credits which he or she can use in the community such as garbage bin upgrade or fee waive. The app uses an assumption that the garbage truck has a bulit-in technology where it can track how much individual household is correctly disposing the garbage. inVision was used to build the prototype of this app.

HappyTribe is volunteering registration app for both hosts and volunteers. The app is a solution to a problem which the users are having hard time trying to find a volunteering event and track changes to the information and capacity of the event.

The core function of the site involves sign-up, mailing, messaging and reservation. My task on this project was to develop the website with other members of the team using git version control tool. The site was developed using Ruby on Rails and various JavaScript library such as hammer.js, google map and jQuery mobile. A lot of feedback between the members went back and forth in a limited time.
